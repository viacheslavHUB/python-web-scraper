# Books Scraper (Production-Ready Web Scraping Project)

A robust Python web scraper that extracts structured data from a multi-page website and exports it into CSV, JSON, and formatted Excel.

Built as a real-world portfolio project demonstrating clean architecture, error handling, and reliable data extraction.

---

## Features

- Multi-page scraping (pagination)
- Retry system (network fault tolerance)
- Clean and structured data extraction
- Export to:
  - CSV
  - JSON
  - Excel (formatted)
- Safe HTML parsing (no fragile chains)
- Automatic URL normalization
- Logging system

---

## Tech Stack

- Python 3.10+
- requests
- BeautifulSoup4
- openpyxl
- urllib3 (retry system)
